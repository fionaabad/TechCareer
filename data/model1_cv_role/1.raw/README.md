# Carpeta 1.raw – datos originales (no versionados)

En esta carpeta van los datasets originales (raw) descargados de Kaggle u otras fuentes.

Estos ficheros **NO se suben al repositorio** porque:

- pueden ser pesados,
- se pueden volver a descargar fácilmente,
- y existe un `.gitignore` que ignora todos los `.csv` fuera de `3.processed`.

### Cómo reproducir los datos

1. Descarga el Dataset 1 (Avishek Majhi) desde:
   https://www.kaggle.com/datasets/avishekmajhi/resume-dataset

   Guarda el CSV aquí como:
   `resumes_dataset1.csv`

2. Descarga el Dataset 2 (Jithin Jagadeesh) desde:
   https://www.kaggle.com/datasets/jithinjagadeesh/resume-dataset

   Guarda el CSV aquí como:
   `resumes_dataset2.csv`

3. Descarga el Dataset 3 (SNEHAAN BHAWAL) desde:
   https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset
   Guarda el CSV aquí como:
   `resumes_dataset3.csv`
